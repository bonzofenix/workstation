# Workstation

This is a full workstation based off pivotal's workstation project.


## Installation

Donwnload [Zip file](https://github.com/bonzofenix/workstation/archive/master.zip) into `~/workstation`


For **OSX**:

Double click installation script.
```sh
osx.command
```

For **Linux**:

```sh
./linux.sh
```
